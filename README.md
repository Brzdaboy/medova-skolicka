# Medová školička Boršice

Landing page pro novou dětskou skupinu Medová školička v Boršicích.

## Nasazení

Stránka je jeden self-contained soubor `index.html` — žádné závislosti ani build step.

### GitHub Pages

1. Nahraj soubory do repozitáře
2. Jdi do **Settings → Pages**
3. Jako source vyber **main** větev, složku **/ (root)**
4. Stránka bude dostupná na `https://<tvůj-username>.github.io/<název-repo>`

### Font (puffin-display)

Font se načítá z Adobe Fonts (`use.typekit.net`). Pokud kit není aktivní, stránka se zobrazí s fallback fontem Fraunces.

## Struktura

```
index.html   ← celá stránka (HTML + CSS + JS + SVG vše inline)
README.md
```
